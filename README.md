## Description
Conwell's Game Of Life simulation

## Environment Setup
- Ruby 2.6.5
- Bundler 1.17.2

install all depedency using bundle in root of directory project.
```
bundle install
```

## Run Test
using rspec & rubocop
```
rake
```

to open the coverage result in browser
```
rake coverage
```

## Run Instruction
To run do
```
ruby lib/main.rb
```
